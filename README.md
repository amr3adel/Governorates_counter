# Egypt Governorates Counter

A lightweight static web app for counting and selecting Egypt's 27 governorates.

## Features

- Live selected governorate counter
- Search by governorate name or region
- Region filters
- Select all visible results
- Static deployment with no build step

## Run Locally

Open `index.html` in a browser, or serve the folder with any static server.

## Deploy

This repo can be deployed directly to Vercel as a static site.
